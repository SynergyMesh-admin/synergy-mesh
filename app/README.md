# Application Directory

This directory contains application-layer components for SynergyMesh.

## Structure

```
app/
└── frontend/
    └── ui/    # Frontend UI components
```

## Components

### Frontend UI (`frontend/ui/`)
The frontend user interface including:
- Dashboard components
- Visualization tools
- User management interfaces

## Development

To develop the frontend:

```bash
cd app/frontend/ui
npm install
npm run dev
```

## Build

To build for production:

```bash
cd app/frontend/ui
npm run build
```
